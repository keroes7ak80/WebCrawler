jsoup 1.3.3 released

2010-Sep-19 jsoup 1.3.3 has been released and is now available for download.

This release:

  • implements Elements.empty() and Elements.remove(). This allows easy element removal, like: doc.select("iframe").remove();
  • adds restricted XHTML output entity option
  • fixes an issue in Entities when unescaping &#36; ("$") (issue #34)
