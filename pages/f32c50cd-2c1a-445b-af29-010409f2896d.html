jsoup version 0.1.2 released

2010-Feb-02 jsoup version 0.1.2 is now available for download. It fixes issue #1, Should treat unknown tags as inline, not block, which improves parsing and cleaning of HTML.

Thanks to François Goldgewicht (francois.goldgewicht@gmail.com) for reporting the issue.

Also thanks to Roberto Rossi for reporting an issue with the documentation.
