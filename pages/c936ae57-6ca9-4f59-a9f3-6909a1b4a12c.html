jsoup 1.5.2 released

2011-Feb-27 jsoup 1.5.2 has been released and is now available for download.

This is a minor release, that primarily corrects a regression bug where the content-type of a document retrieved using Jsoup.connect(String url) may not be correctly detected if specified in a meta tag.

Fixes and improvements

  • Fixed issue with selector parser where some boolean AND + OR combined queries (e.g.meta[http-equiv], meta[content]) were being parsed incorrectly as OR only queries (e.g. former as meta, [http-equiv], meta[content])
  • Fixed issue where a content-type specified in a meta tag may not be reliably detected, due to the above issue.
  • Updated Element.text() and Element.ownText() methods to ensure <br> tags output as whitespace.
  • Tweaked Element outer HTML method to not generate initial newline on first output element.
